# Phase 6 - Logs

This folder contains Bash scripts for monitoring and analyzing system logs:

- 01-log-basics.sh: List logs in /var/log
- 02-auth-log-monitor.sh: Show last 10 failed login attempts
- 03-bruteforce-detector.sh: Detect repeated failed SSH login attempts by IP
- 04-error-summary.sh: Show last 20 system errors
